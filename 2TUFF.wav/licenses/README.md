# Bundled font licenses

2TUFF.wav embeds glyph bitmaps from two fonts directly in `src/font_plex.h`.
Both are distributed under the SIL Open Font License, Version 1.1. The full
license text for each is included here, as the OFL requires when redistributing
the fonts (or anything derived from them).

| Font            | Author / Copyright                                   | License file                |
|-----------------|------------------------------------------------------|-----------------------------|
| IBM Plex Mono   | Copyright © 2017 IBM Corp.                            | `IBM-Plex-Mono-OFL.txt`     |
| Pixelify Sans   | Copyright 2021 The Pixelify Sans Project Authors      | `Pixelify-Sans-OFL.txt`     |

These licenses cover the bundled fonts only.
