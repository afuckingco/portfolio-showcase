# Portfolio Showcase
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)  [![CI](https://github.com/afuckingco/portfolio-showcase/actions/workflows/ci.yml/badge.svg)](https://github.com/afuckingco/portfolio-showcase/actions)


Consolidated collection of data science / ML portfolio projects, each in its own subdirectory with independent history preserved via git subtree.

## Projects

- **kopikita/** — Cafe sales analytics & forecasting pipeline (EDA, Prophet forecasting, RFM segmentation)
- **air-quality-gru-reset/** — Air Quality (PM2.5) prediction with periodic GRU reset
- **sentiment-streaming/** — Real-time streaming sentiment analysis for tourist comments (Redis)
- **fraud-detection/** — Fraud detection pipeline (SMOTE + XGBoost, FastAPI, Streamlit dashboard)
- **bali-tourism-mlops/** — Tourist arrivals prediction with LSTM (periodic partial reset)
- **dvwa-portfolio/** — Professional portfolio for DVWA security research

Each subdirectory retains its own README, LICENSE, and CI configuration.
