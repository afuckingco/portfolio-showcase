# portfolio-showcase

15+ projects across ML, Security, Web, Embedded, GIS — unified showcase and documentation hub.

## Stack

- Python 3.11+
- PyTorch / scikit-learn
- See `requirements.txt` or `pyproject.toml`

## Quick Start

```bash
git clone https://github.com/afuckingco/portfolio-showcase.git
cd portfolio-showcase
pip install -r requirements.txt
```

## CI

GitHub Actions — lint (ruff), build (py_compile). Every push triggers a run.
